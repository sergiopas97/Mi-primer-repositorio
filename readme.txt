Git y GitHub son herramientas esenciales para el desarrollo de software colaborativo.
 Git es unsistema de control de versiones que permite a los desarrolladores llevar un registro de cambios en el có-digo y 
colaborar en proyectos de manera eficiente. 
GitHub es una plataforma en línea que proporcionaun alojamiento para repositorios de Git,
 así como una serie de características adicionales para facilitar lacolaboración en proyectos. 
En esta práctica, repasaremos los conceptos e instrucciones básicas para Git yGithub y gestionar tus propios repositorios.
